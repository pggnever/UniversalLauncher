# 🚀 Universal Launcher (v1.0)

[English](#english) | [Русский](#русский)

---

## English

**Universal Launcher** is a lightweight utility for gamers to automate application sequences (Game + Anti-Cheat + Discord + Browser).

### ✨ Features
- **🔍 Smart Detection**: Launches the rest of the sequence when the main app (e.g., Faceit AC) starts.
- **⏳ Sequential Startup**: Set delays (seconds) between apps.
- **🧹 Auto-Close**: Automatically closes background apps when you exit the game.
- **🌍 Multi-language**: Full RU/EN support.
- **🥷 Tray Mode**: Runs quietly in the system tray.

### 📦 Installation
1. Download `UniversalLauncher.exe` from the [Releases](https://github.com/твой_ник/UniversalLauncher/releases) section.
2. Run as Administrator.

## Русский

**Universal Launcher** — легкая утилита для геймеров, автоматизирующая запуск игровых связок (Игра + Античит + Discord + Браузер).

### ✨ Особенности
- **🔍 Умное обнаружение**: Запускает всю цепочку, как только обнаруживает старт главной программы (например, Faceit AC).
- **⏳ Очередность**: Настраиваемые задержки в секундах между запусками.
- **🧹 Авто-закрытие**: Закрывает второстепенные программы после выхода из игры.
- **🌍 Мультиязычность**: Интерфейс на RU и EN.
- **🥷 Трей-режим**: Работает в фоне и не мешает игре.

### 📦 Установка
1. Скачайте `UniversalLauncher.exe` в разделе [Releases](https://github.com/pggnever/UniversalLauncher/releases).
2. Запускайте от имени администратора.
